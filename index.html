Why HTML for Cybersecurity?

Understanding HTML helps security practitioners:

  • Find injection points (forms & inputs)
  • Examine DOM structure for XSS and logic bugs
  • Build simple tools and reports
XSS
Cross-Site Scripting — injecting scripts into web pages.
CSRF
Cross-Site Request Forgery — manipulating authenticated actions.
